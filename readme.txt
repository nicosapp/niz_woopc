Create quickly and easily a beautiful dynamic Woocommerce products carousel for your e-commerce website.

== Description ==

With Niz Products Carousel for Woocommerce include products carousel in your pages.

Use any of the Woocommerce shortcodes to create your products carousel.

Customize your carousel with the following attributes:
* Number of products in the carousel
*Number of products visible at once in the carousel
* Autoplay
* Pause
* Loop
* Navigations buttons
* Dots buttons
* Speed of the carousel
* Delay between swipes

With one shortcode for each carousel, create as many carousels as you want in the same page and/or different pages!

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Use the Settings->Plugin Name screen to configure the plugin

== Frequently Asked Questions ==

= Is this carousel working with Woocommerce product related shortcodes? =

Yes, you can create a Niz carousel with any defined Woocommerce product shortcode!

= Can I create multiple product carousels on the same page? =

Yes, you can create several carousels. Thanks to the Niz shortcode generator, create your custom shortcode and copy-paste it wherever you want on your pages.
